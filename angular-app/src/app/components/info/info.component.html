Käyttöohjeet

Työkalun lainaaminen:

1. Paina "LAINAA" nappulaa
2. Näytä henkilökorttiasi lukijalle
3. Näytä lainattavaa tavaraa lukijalle
4. Valitse alue minne lainaat tavaran
Tavara on nyt lainattu sinulle.
Tieto lainauksesta ilmestyy lainauslistalle.

Tavaran palauttaminen:

1. Paina "PALAUTA" nappulaa
2. Näytä palautettavaa tavaraa lukijalle
Tavara on nyt palautettu.
Kuka tahansa voi palauttaa lainatun tavaran.

Järjestelmän hallinta:

1. Avaa asetukset
2. Lue opettajan henkilökortti
3. Lisää tai poista asioita järjestelmästä.

Järjestelmän tiedot

Käyttöliittymä: Angular + Electron
Palvelin: Express + Node
Tietokanta: MongoDb
RFID lukija: Arduino Uno R3 + PN532 RFID/NFC Shield
