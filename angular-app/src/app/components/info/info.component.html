OHJEET KÄYTTÄJÄLLE

Tavaran lainaus:
Paina LAINAA TAVARA nappulaa -->
Näytä henkilökorttiasi lukijalle -->
Näytä lainattavaa tavaraa lukijalle-->
Tavara on nyt lainattu sinulle
Nimesi ja lainattu tavara sekä tavaran lainattuna kulunut aika ilmestyy pääsivun oikealla puolella olevalle listalle.

Tavaran palautus:
Paina PALAUTA TAVARA nappulaa -->
Näytä palautettavaa tavaraa lukijalle -->
Tavara on nyt palautettu.
Kuka tahansa voi palauttaa lainatun tavaran.

HALLINTA nappulasta pääsee opettajan oikeuksilla ylläpitäjän työkaluihin. Täällä tapahtuu käyttäjän/tavaran lisääminen/poistaminen sekä ohjelman sulkeminen.

System components:
Frontend = Angular 2+
Backend = Express.js + Node.js
Database = MongoDb
Scanner = Arduino Uno + RFID scanner
